# formatERC1271Hash

Formats the hash to be signed for ERC1271 signature validation.

## Usage

```solidity copy
bytes32 formattedHash = instance.formatERC1271Hash({validator: address(instance.defaultValidator), hash: hash});
```

## Parameters

### instance

- Type: [`AccountInstance`](/modulekit/glossary/types.mdx#accountinstance)

The instance of the account used for testing.

### validator

- Type: `address`

The address of the validator contract.

### hash

- Type: `bytes32`

The hash to format for ERC1271 signature validation.

## Returns

### formattedHash

- Type: `bytes32`

The formatted hash to sign for ERC1271 signature validation.
