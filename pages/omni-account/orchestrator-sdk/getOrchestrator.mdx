# getOrchestrator

Get the client to interact with the Orchestrator.

## Usage

```typescript copy
const orchestrator = getOrchestrator('ORCHESTRATOR_API_KEY')
```

## Parameters

### apiKey

- Type: `string`

The API key to use for the orchestrator.

## Returns

### orchestrator

- Type: `Orchestrator`

The client class to interact with the Orchestrator.
