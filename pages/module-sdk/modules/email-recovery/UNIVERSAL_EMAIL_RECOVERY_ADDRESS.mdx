# UNIVERSAL_EMAIL_RECOVERY_ADDRESS

Returns the address of the universal email recovery module.

For the latest deployment addresses and supported networks, see the [ZK Email docs](https://docs.zk.email/account-recovery/deployed-contracts).


## Usage

```typescript
const moduleAddress = UNIVERSAL_EMAIL_RECOVERY_ADDRESS
```

## Parameters

None

## Returns

### moduleAddress

- Type: [`Address`](https://viem.sh/docs/glossary/types#address)

The address of the universal email recovery module.
