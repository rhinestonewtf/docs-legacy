# OmniAccount 

**An offchain service for coordinating cross-chain intents**

The Rhinestone Orchestrator allows you to coordinate cross-chain intents in order to achieve chain abstraction.

## Orchestrator SDK

**A TypeScript library for using the Rhinestone Orchestrator**

The Orchestrator SDK is a TypeScript library that enables convenient interactions with the Orchestrator as well as helper utilities around the required modules for making cross-chain intents.
